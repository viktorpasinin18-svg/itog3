# Random Quote Generator

**Автор:** Пашинин Виктор

## Описание
Random Quote Generator — это десктопное приложение с графическим интерфейсом на Python (tkinter), которое позволяет:
- Получать случайные мотивирующие или философские цитаты
- Фильтровать цитаты по автору или теме
- Сохранять историю просмотренных цитат
- Добавлять собственные цитаты в коллекцию

Все данные сохраняются в JSON-файлы, что позволяет не терять историю и добавленные цитаты после перезапуска программы.

## Возможности

### 🎲 Генерация случайных цитат
- Один клик — новая случайная цитата из доступной коллекции
- Поддержка кириллицы и любых языков
- Красивое форматирование с указанием автора и темы

### 🔍 Фильтрация
- **По автору** — показывать цитаты только выбранного автора
- **По теме** — фильтрация по категориям (мотивация, философия, жизнь, успех и др.)
- **Комбинированная фильтрация** — можно фильтровать одновременно по автору И теме
- **Сброс фильтров** — кнопка для быстрого возврата ко всем цитатам
- Индикация количества найденных цитат после фильтрации

### 📜 История цитат
- Автоматическое сохранение каждой сгенерированной цитаты
- Отображение времени генерации (дата и время)
- Показ последних 20 цитат в хронологическом порядке (сначала новые)
- Сохранение истории в файл `history.json`

### ➕ Добавление новых цитат
- Интуитивно понятная форма добавления
- Валидация ввода — все поля обязательны для заполнения
- Проверка на пустые строки с выводом сообщения об ошибке
- Автоматическое обновление списков фильтров после добавления

### 💾 Сохранение данных
- Все цитаты хранятся в `quotes.json`
- История генераций — в `history.json`
- Автоматическая загрузка при запуске
- Автоматическое сохранение при изменениях

### 🛡️ Обработка ошибок
- Защита от генерации при отсутствии цитат
- Предупреждения при пустых результатах фильтрации
- Проверка на пустые поля при добавлении цитат
- Корректная работа даже если файлы данных отсутствуют (создаются автоматически с цитатами по умолчанию)

### 🎨 Пользовательский интерфейс
- Понятная структура с группировкой элементов
- Цветовые индикаторы для кнопок и статусов
- Прокрутка для длинных списков (история)
- Адаптивный дизайн с возможностью изменения размера окна
